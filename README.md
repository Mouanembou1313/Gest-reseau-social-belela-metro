# Gest Réseau Social - Application Mobile

Une application mobile de réseau social développée avec React Native et TypeScript.

## 📋 Prérequis

- Node.js >= 20
- npm ou yarn
- React Native CLI
- Android Studio (pour Android)
- Xcode (pour iOS)

## 🚀 Installation

\`\`\`bash
# Cloner le dépôt
git clone https://github.com/Mouanembou1313/Gest-reseau-social-belela-metro.git
cd Gest-reseau-social-belela-metro

# Installer les dépendances
npm install
\`\`\`

## 🎯 Lancer l'application

### Android
\`\`\`bash
npm run android
\`\`\`

### iOS
\`\`\`bash
npm run ios
\`\`\`

### Démarrer le serveur de développement
\`\`\`bash
npm start
\`\`\`

## 📁 Structure du projet

\`\`\`
src/
├── screens/          # Écrans de l'app
├── navigations/      # Configuration de navigation
├── components/       # Composants réutilisables
├── assets/           # Images et ressources
